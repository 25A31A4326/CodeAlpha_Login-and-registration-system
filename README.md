# 🔐 Login and Registration System in C++

## 📌 Project Title
Login and Registration System

---

## 📝 Project Overview

The Login and Registration System is a simple C++ application that allows users to create an account and log in using their credentials. The system stores user information in a file and verifies login details during authentication.

This project helps in understanding file handling, functions, conditional statements, loops, and user authentication concepts in C++.

---

## 🎯 Objective

- To register new users
- To store user information securely in a file
- To allow registered users to log in using a username and password
- To verify user credentials and provide access only to authorized users

---

## ⚙️ Key Concepts Used

- Functions
- File Handling
- Strings
- Conditional Statements (if-else)
- Loops
- User Authentication

---

## 📊 Algorithm / Working Steps

### 📝 Registration

1. Select the Register option
2. Enter username and password
3. Store user details in a file
4. Display registration success message

### 🔑 Login

1. Select the Login option
2. Enter registered username and password
3. Compare entered details with stored data
4. If details match, login is successful
5. Otherwise, display an error message
